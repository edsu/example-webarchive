Self Hosted Web Archives

This is a Web 1.0 website that provides an example of how to self-host your own web archives using plain old HTML.

The web archives were created with Webrecorder tools, exported as WACZ files and are embedded here using the <replay-web-page> Web component.

Check it out on GitHub if you want to see how it works, and send us a PR adding another web archive if you like!
